# 跨境文案撰写 (ML Copywriter)

Mercado Libre 多站点（墨西哥、巴西、哥伦比亚、智利、阿根廷、乌拉圭）Listing 文案优化。

## 适用场景

- 用户需要撰写/优化 Mercado Libre 产品标题
- 用户需要写西班牙语或葡萄牙语产品描述
- 用户需要提取关键词做 SEO 优化
- 用户需要翻译中文/英文产品信息到西语/葡语
- 用户需要对比竞争对手的 Listing 文案

## 标题优化规则

### ML 标题格式标准：
```
[品牌] + [核心产品名] + [关键特性] + [规格/尺寸] + [适用场景/对象]
```

### 字符限制
- ML Mexico: 最长 60 字符（建议 45-55）
- ML Brazil: 最长 60 字符
- ML Colombia/Chile/Argentina: 最长 60 字符

### 关键词优先级
1. 核心产品词（fundas para celular, aretes, collares...）
2. 材质/特性（titanio, acero inoxidable, silicona, ajustable...）
3. 适用对象（para mujer, para hombre, para niños, unisex...）
4. 数量/包装（Pack x 5, Set de 10, Lote...）
5. 卖点（Original, Premium, Resistente, Impermeable...）

### 禁用词
- El mejor, #1, más vendido, garantizado, perfecto（夸大用语）
- 价格词（barato, económico, gratis）
- 比较词（mejor que...）
- 虚假承诺（resultados en 24h）

## 描述结构

### 西班牙语（墨西哥/哥伦比亚/智利/阿根廷/乌拉圭）
```
[开篇：痛点/需求场景]
¿Buscas [producto] de alta calidad? [Breve introducción]

[特性列表：图标+短句]
• [Material] de primera calidad - [explicación breve]
• [Diseño] ergonómico - [beneficio]
• [Tamaño/Peso] ideal para [uso]
• [Color/Estilo] disponible en [variantes]

[规格表]
Dimensiones: [X cm x Y cm x Z cm]
Material: [material]
Peso: [X g]
Color: [colores]
Incluye: [contenido del paquete]

[使用场景]
Perfecto para [uso 1], [uso 2] y [uso 3].

[cuidado/mantenimiento si aplica]

[garantía opcional]
```

### 葡萄牙语（巴西）
```
[开篇]
Procurando [produto] de qualidade? [Introdução breve]

[特性]
• [Material] de primeira qualidade - [explicação]
• [Design] ergonômico - [benefício]
• [Tamanho] ideal para [uso]
• [Cor/Estilo] disponível em [variantes]

[especificações]
Dimensões: [X cm x Y cm x Z cm]
Material: [material]
Peso: [X g]
Cor: [cores]
Inclui: [conteúdo]

[uso]
Perfeito para [uso 1], [uso 2] e [uso 3].

[garantia opcional]
```

## 国家站点差异化

| 站点 | 语言 | 价格符号 | 常用搜索词特征 |
|------|------|---------|---------------|
| Mexico | ES | MXN $ | 英文借词多（bag, case, set） |
| Brazil | PT-BR | R$ | 葡语拼写特有（boneca→bonecas） |
| Colombia | ES | COP $ | 用词偏保守（bolso vs cartera） |
| Chile | ES | CLP $ | 用词偏当地（polera vs camiseta） |
| Argentina | ES | ARS $ | 独特俚语多 |
| Uruguay | ES | UYU $ | 接近阿根廷用词 |

## 关键词提取方法

1. 从 ML 搜索结果页提取标题高频词
2. 从同类目 Top Seller 标题提取关键词组合
3. 使用 AMZ123 热词表验证搜索量
4. 检查 ML 搜索下拉建议词

## 注意事项

- 巴西葡萄牙语和葡萄牙葡语差异大，务必用 PT-BR
- ML Argentina 和 Uruguay 有独特俚语，避免用墨西哥/哥伦比亚用词
- 标题不要堆砌关键词（keyword stuffing），ML 会降权
- 描述建议 500-1000 字符，太长用户不看完
- 重点突出 shipping（envio gratis, envio rápido）如果适用
